Database middleware

The below functions represent functions applied to database objects before they are sent to (setters), and retrieved from (getters) the database. Defaults are applied if no other value is set, and validator functions are applied both at database level and at the front-end, in forms and in other areas where the corresponding field directives are used.

All functions are automatically applied to field types set on schemas, however currently a restart is required in order to ensure this happens.
